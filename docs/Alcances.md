## Mini GitHub - Alcances

El proyecto incluye las siguientes funcionalidades:

- **Autenticación:** Registro, login con email/contraseña, autenticación OIDC/SSO mediante Keycloak (con federación de GitHub y Google) y recuperación de contraseña.
- **Gestión de usuarios y equipos:** Roles básicos (Owner, Developer, Reporter) para control de acceso a repositorios.
- **Repositorios Git:** Crear repositorios públicos y privados, subir/descargar archivos, navegación de estructura de carpetas, visualización de contenido y fork de repositorios públicos.
- **Issues:** Crear, comentar, etiquetar, asignar y cerrar issues dentro de un repositorio.
- **Búsqueda:** Búsqueda de repositorios por nombre y usuarios por username.
- **Colaboración:** Dar/quitar estrellas a repositorios, gestión de colaboradores con roles por repositorio.
- **Notificaciones por email:** Alertas en eventos clave como nuevos issues, comentarios e invitaciones a colaboradores.
- **API REST documentada:** Endpoints documentados con Swagger UI.
